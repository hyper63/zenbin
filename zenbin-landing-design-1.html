API

Simple, RESTful, Obvious

One endpoint to create, one to retrieve. No auth headers, no tokens. Just POST your HTML.

Create a page POST
POST /v1/pages/{id} Content-Type: application/json { "html": "<h1>Hello World</h1>", "markdown": "# Hello World", "title": "My Page", "auth": { "password": "secret" } }
Response JSON
{ "id": "my-page", "url": "https://zenbin.io/p/my-page", "raw_url": "https://zenbin.io/p/my-page/raw", "markdown_url": "https://zenbin.io/p/my-page/md" } # That's it. No auth headers. No tokens. Just POST.
